Modele arhitecturale
        Three-Tier Arhitecture
            presentation tier   - nivel utilizator
            application tier    - nivel aplicatie
            data tier           - nivel server de date
        Two-Tier Arhitecture 
            presentation tie    - nivel utilizator
            data tier           - nivel server de date

Model relational
        fondat pe algebra relationala
        colectie de relatii ~ tabela cu valori  
        relatie = tabel
        atribut = nume + tip
    
Model nerelational
        scalabilitate orizontala / flexibilitate
        nu se bazeaza pe coloane    

Obiect  -   structura de date   
        -   cel mai comun obiect : tabel

Tabel   -   conceptul unei baze de date
        -   contine : coloane, randuri  
                - coloane: definesc tipurile de date
                - randuri: reprezinta datele din tabel
